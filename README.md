## baxterstreet

### Setup oh-my-zsh

#### Install baxterstreet plugin
```zsh
pushd $ZSH/custom/plugins && \
  git clone git@github.com:johnlayton/baxterstreet.git baxterstreet && \
  popd || echo "I'm broken"
```
```zsh
plugins=(... kenv)
```

### Setup other

```zsh
pushd $HOME && \
  git clone git@github.com:johnlayton/baxterstreet.git .baxterstreet && \
  popd || echo "I'm broken"
```

```zsh
source ~/.baxterstreet/baxterstreet.plugin.zsh
```
